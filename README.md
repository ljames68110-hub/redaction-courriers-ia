\# ✉️ Rédaction de Courriers IA  

Application FastAPI permettant de générer automatiquement des courriers administratifs personnalisés grâce à l’IA, avec gestion des détenus, RAG (FAISS), et API REST complète.



\---



\## 🚀 Fonctionnalités principales



\- Génération automatique de courriers (administratif, personnel, formel, etc.)

\- Gestion des détenus (création, mise à jour, sauvegarde)

\- Moteur IA basé sur Transformers

\- Système RAG avec FAISS pour améliorer la pertinence

\- API REST documentée via Swagger (`/docs`)

\- Architecture modulaire et extensible
